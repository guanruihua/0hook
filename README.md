# 0hook
>
> 封装日常开发常使用的hook
>
### useBoolean
>
> boolean 状态
>
### useCount
>
> 数字状态
>
### useMap
>
> Map状态
>
### useSetState
>
> 当state为对象时候, 只修改对象一个值, 就只传对应值即可, 不用全传输
> 类似 `Class Component` 的 `this.setState`
>
### useObject
>
> object 状态, 类似 `useSetState`的更加多方法版
>
### useStorage
>
> 字符串状态, 使用Storage 状态缓存
> 有`useLocalStorage` 和 `useSessionStorage`
>
### useUpdate
>
> 更新组件
